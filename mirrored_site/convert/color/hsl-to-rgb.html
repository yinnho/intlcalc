HSL to RGB conversion formula

When 0 ≤ H < 360, 0 ≤ S ≤ 1 and 0 ≤ L ≤ 1:

C = (1 - |2L - 1|) × S

X = C × (1 - |(H / 60°) mod 2 - 1|)

m = L - C/2

(R,G,B) = ((R'+m)×255, (G'+m)×255,(B'+m)×255)
